# Madrid Open Static Site

This repository contains only the static front-end files for the Madrid Open player hub.
Data collection and snapshot generation happen outside this public repository.

## Files

- `index.html`: page structure
- `styles.css`: page styles
- `app.js`: browser logic
- `data/latest.json`: static data snapshot consumed by the page

Open `index.html` in a browser, or publish the repository with GitHub Pages.
The GitHub Pages workflow deploys the checked-in static files only; it does not fetch or regenerate data.
